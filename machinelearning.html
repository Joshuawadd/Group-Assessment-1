Machine Learning


What is machine learning?

A subset of the field of Artificial Intelligence, Machine Learning is the process of training a computer to teach itself how to execute a complicated task, without directly programming how to execute the task. This is done through supplying the computer with a large amount of data for it to analyse, learn from, and ultimately predict how to best achieve a desirable outcome. This has rapidly become an increasingly popular approach to solving tasks with advancements in computer hardware allowing Machine Learning to become more viable.


How can machine learning contribute to society?

Machine Learning allows computers to solve new problems that have typically required human involvement to solve efficiently. Some examples of this include:

  • Self-Driving Vehicles
  • Medical Diagnosis
  • Fraud Detection
  • Facial, Speech, and Handwriting Recognition
  • Targeted advertising
